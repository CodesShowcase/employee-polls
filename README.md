# Employee Polls Project

This project was bootstrapped with [Create React App][1].

## Description

This project is about an employee polls board. Users can create questions with two voting options, answer their own and other usersâ€™ questions and have the possibility to watch the current stats for a single question and the overall stats of all employees.

## Instructions

The project is a react redux project
- npm install
- npm start (runs the app on your local machine with the standard port 3000)
- npm run build (builds the app for production mode in the build folder)
- npm test (runs all the available tests)
